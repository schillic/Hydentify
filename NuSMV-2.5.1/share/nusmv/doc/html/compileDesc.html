The compile package

Compilation of NuSMV input language into BDD.

By Marco Roveri, Emanuele Olivetti

  • Information for programmers
  • Information for developers sorted by function
  • Information for developers sorted by file
This package contains the compiler of NuSMV code into BDD. It works on a flattened/instantiated structure. Performs the checks on the parse tree and fills a layer (when requested) and the symbol table.
