  bmc_setup - Builds the model in a Boolean Epression format.

   --------------------------------------------------------------------------

   bmc_setup [-h] | [-f]

   You must call this command before use any other bmc-related command. Only
   one call per session is required.
   Command options:

   -f
           Forces the BMC model to be built.
